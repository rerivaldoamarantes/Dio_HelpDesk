# Dio_HelpDesk
Projeto de uma modelagem para BD MySql de uma aplicação de serviço de chamados para serviços nos veículos da empresa utilizados por diversos usuários, incluindo avaliação de chamados e veículos de propriedade da empresa e de terceiros (locado). Cada usuário abre um chamado que pode gerar uma ordem de serviço ou ser arquivado, conforme responsável.
